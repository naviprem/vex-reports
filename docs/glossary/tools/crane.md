# crane

[[registry|Registry]] Swiss-army knife from Google's go-containerregistry project — `curl` for container images. Fetches digests, [[manifest|manifests]], [[image-config|configs]], and [[layer|layers]]; copies images between registries; all daemon-free.

Pilot use: pinning the image [[tag-vs-digest|digest]] so every VEX claim binds to an immutable artifact.
