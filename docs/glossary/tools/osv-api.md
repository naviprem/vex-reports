# OSV API

Free HTTPS API returning any advisory in [[osv|OSV]] schema: `https://api.osv.dev/v1/vulns/<id>` — works for [[cve|CVE]], [[ghsa|GHSA]], GO-, RUSTSEC- IDs alike, with the [[alias|aliases]] field included.

The pilot's rung F4 source for advisory details and preconditions, and F1's source for alias resolution.
