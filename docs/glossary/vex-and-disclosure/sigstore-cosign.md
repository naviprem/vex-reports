# Sigstore / cosign

Open-source signing ecosystem for software artifacts. `cosign` signs [[container-image|container images]] and attaches attestations ([[sbom|SBOMs]], [[vex|VEX]] documents) to them in the [[registry]].

Phase 2 of the parent PRD: sign published VEX reports with enterprise-managed keys (AWS KMS via cosign) and document the customer verification procedure.
